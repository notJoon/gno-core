---
name: commit_title_only
description: User prefers commit messages with title only, no body/description
type: feedback
---

Commit messages should contain only the title line (e.g. `feat(gnolang): add shouldInlineArray and copyArrayInline helpers`), without a body or description.

**Why:** User prefers concise commits without verbose descriptions.
**How to apply:** When creating git commits, use only a single-line commit message. Do not add multi-line body, bullet points, or Co-Authored-By lines.
